문제1. 약수 구하기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120897

문제 설명

정수 n이 매개변수로 주어질 때, n의 약수를 오름차순으로 담은 배열을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ n $\leq$ 10000.

입출력 예

n result
24 [1,2,3,4,6,8,12,24]
29 [1,29]

코드 및 설명

#include <string>
#include <vector>

using namespace std;

vector<int> solution(int n) {
    vector<int> answer;
    
    for (int i = 1; i <= n; i++) {
        if (n % i == 0) {
            answer.push_back(i);
        }
    }
    
    return answer;
}

for문을 사용하여 n 이하의 모든 양의 정수에 대해서 나눴을 때 나머지가 0이면 vector에 push back하였다.

문제2. 369게임

url: https://school.programmers.co.kr/learn/courses/30/lessons/120891

문제 설명

머쓱이는 친구들과 369게임을 하고 있습니다. 369게임은 1부터 숫자를 하나씩 대며 3, 6, 9가 들어가는 숫자는 숫자 대신 3, 6, 9의 개수만큼 박수를 치는 게임입니다. 머쓱이가 말해야하는 숫자 order가 매개변수로 주어질 때, 머쓱이가 쳐야할 박수 횟수를 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • 1 $\leq$ order $\leq$ 1,000,000

입출력 예

numbers result
3 1
29423 2

코드 및 설명

#include <string>
#include <vector>

using namespace std;

int solution(int order) {
    int answer = 0;
    string str_order = to_string(order);
    for (auto a : str_order) {
        if (a == '3' || a == '6' || a == '9')
            answer++;
    }
    return answer;
}

우선 int형인 order를 string으로 바꾼 후, string의 각 character에 대해 3 또는 6 또는 9 일 경우에 answer의 값을 1씩 증가시켰다.
int형을 string으로 바꿀 때는 to_string, string을 int형으로 바꿀 때는 stoi.

다른 사람 풀이를 보니까 각 자리수를 비교하기 위해 while문을 돌면서 10으로 나눈 나머지를 확인한 후 10으로 나눠서 각 자리수를 확인하는 방법이 있었다.

while (order > 0) {
  int tmp = order % 10;
  if (tmp != 0 && (tmp % 3)== 0)
    answer++;
  order /= 10;
}

문제3. 주사위의 개수

url: https://school.programmers.co.kr/learn/courses/30/lessons/120845

문제 설명

머쓱이는 직육면체 모양의 상자를 하나 가지고 있는데 이 상자에 정육면체 모양의 주사위를 최대한 많이 채우고 싶습니다. 상자의 가로, 세로, 높이가 저장되어있는 배열 box와 주사위 모서리의 길이 정수 n이 매개변수로 주어졌을 때, 상자에 들어갈 수 있는 주사위의 최대 개수를 return 하도록 solution 함수를 완성해주세요.

제한 사항

  • box의 길이는 3입니다.
  • box[0] = 상자의 가로 길이
  • box[1] = 상자의 세로 길이
  • box[2] = 상자의 높이 길이
  • 1 $\leq$ box의 원소 $\leq$ 100
  • 1 $\leq$ n $\leq$ 50
  • n $\leq$ box의 원소
  • 주사위는 상자와 평행하게 넣습니다.

입출력 예

box n result
[1,1,1] 1 1
[10,8,6] 3 12

코드 및 설명

#include <string>
#include <vector>

using namespace std;

int solution(vector<int> box, int n) {
    int answer = 0;
    answer = (box[0]/n) * (box[1]/n) * (box[2]/n);
    return answer;
}

가로, 세로, 높이를 모두 n으로 나눠서 나오는 몫(정수)를 곱했다.
괄호를 넣지 않은 경우 순서대로 연산을 해서 그런가 중간에 나누어 떨어져서인지 테스트 2가 실패로 떴다.

문제4. 외계행성의 나이

url: https://school.programmers.co.kr/learn/courses/30/lessons/120834

문제 설명

우주여행을 하던 머쓱이는 엔진 고장으로 PROGRAMMERS-962 행성에 불시착하게 됐습니다. 입국심사에서 나이를 말해야 하는데, PROGRAMMERS-962 행성에서는 나이를 알파벳으로 말하고 있습니다. a는 0, b는 1, c는 2, …, j는 9입니다. 예를 들어 23살은 cd, 51살은 fb로 표현합니다. 나이 age가 매개변수로 주어질 때 PROGRAMMER-962식 나이를 return하도록 solution 함수를 완성해주세요.

제한 사항

  • age는 자연수입니다.
  • age $\leq$ 1,000
  • PROGRAMMERS-962 행성은 알파벳 소문자만 사용합니다.

입출력 예

age result
23 “cd”
51 “fb”
100 “baa”

코드 및 설명

#include <string>
#include <vector>

using namespace std;

string solution(int age) {
    string answer = "";
    while (age > 0) {
        int i = age % 10;
        char ch_age = 'a' + i;
        answer = ch_age + answer;
        age /= 10;
    }
    
    return answer;
}

이 문제도 각 자릿수 숫자들을 확인해야하므로 위에서처럼 to_string을 사용할까 했는데 숫자를 알파벳으로 바꿀 때 다시 int형이 필요해서 그냥 숫자인채로 사용했다.
1의 자릿수부터 구해서 그에 맞는 알파벳을 구한 후, 이를 answer의 앞에 더했다. 다음엔 10의 자릿수이므로 앞에 계속 알파벳을 더하는 형식으로 했다.
이 문제도 다른 사람 풀이가 엄청 간단했다!!

string answer = to_string(age);
for (auto& v: answer) {
  v += 'a' - '0';
}
return answer;

와 이건 생각도 못 했다. 생각해보면 age를 string으로 바꾼 후 거기에 ‘0’을 빼면 그 int값이 나올 것이고, 거기에 ‘a’를 더하면 해당 알파벳이 나올 것이다. 대단하다..

문제5. 인덱스 바꾸기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120895

문제 설명

문자열 my_string과 정수 num1, num2가 매개변수로 주어질 때, my_string에서 인덱스 num1과 인덱스 num2에 해당하는 문자를 바꾼 문자열을 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • 1 < my_string의 길이 < 100
  • 0 $\leq$ num1,num2 < my_string의 길이
  • my_string은 소문자로 이루어져 있습니다.
  • num1$\neq$ num2

입출력 예

my_string num1 num2 result
“hello” 1 2 “hlelo”
“I love you” 3 6 “I l veoyou”

코드 및 설명

#include <string>
#include <vector>

using namespace std;

string solution(string my_string, int num1, int num2) {
    string answer = "";
    char tmp = my_string[num1];
    my_string[num1] = my_string[num2];
    my_string[num2] = tmp;
    answer = my_string;
    return answer;
}

tmp에 문자 하나를 잠시 저장해두고 서로의 위치를 바꿨다.
이것도 다른 사람 풀이를 확인하니까 swap 함수를 사용하는 게 제일 깔끔했다.
algorithm 라이브러리에 있는 함수인줄 알았는데 STL에 있는 함수였다.. 정말 기본적인 함수도 잘 모른다는 걸 알게됐다..

총평

우선 오늘은 문제4에서의 다른 사람 풀이가 가장 인상적이었다.
문제5에서 swap을 사용하는 것도 정말 기본 STL에 있는 함수들도 잘 기억을 못 한다는 걸 깨닫고 다시 공부하기로 결심!
