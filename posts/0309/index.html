문제1. 영어가 싫어요

url: https://school.programmers.co.kr/learn/courses/30/lessons/120894

문제 설명

영어가 싫은 머쓱이는 영어로 표기되어있는 숫자를 수로 바꾸려고 합니다. 문자열 numbers가 매개변수로 주어질 때, numbers를 정수로 바꿔 return 하도록 solution 함수를 완성해 주세요.

제한 사항

  • numbers는 소문자로만 구성되어 있습니다.
  • numbers는 “zero”, “one”, “two”, “three”, “four”, “five”, “six”, “seven”, “eight”, “nine” 들이 공백 없이 조합되어 있습니다.
  • 1 $\leq$ numbers의 길이 $\leq$ 50
  • “zero"는 numbers의 맨 앞에 올 수 없습니다.

코드 및 설명

def solution(numbers):
    str_number = ["zero", "one", "two", "three", "four", "five", "six", "seven", "eight", "nine"]
    for i in str_number:
        numbers = numbers.replace(i, str(str_number.index(i)))
    return int(numbers)

이 문제는 살짝 어려웠다. 문제 배점도 4점이었다!
우선 이 문제에서 헤맸던 부분은 문자열.replace가 replace를 한 string을 return한다는 점이다.
이걸 모르고 그냥 numbers.replace()로만 써서 왜 답이 안 나오지? 했는데 string을 return해서 그걸 다시 문자열에 넣어야됐었다..
이 문제도 다른 사람 풀이를 보니까 enumerate을 사용하면 좋을 듯!

문제2. 몫 구하기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120805

문제 설명

정수 num1, num2가 매개변수로 주어질 때, num1을 num2로 나눈 몫을 return 하도록 solution 함수를 완성해주세요.

제한 사항

  • 0 $\leq$ num1 $\leq$ 100
  • 0 $\leq$ num2 $\leq$ 100

코드 및 설명

def solution(num1, num2):
    return num1 // num2

문제6. 나머지 구하기

문제 설명

정수 num1, num2가 매개변수 주어질 때, num1를 num2로 나눈 나머지를 return 하도록 solution 함수를 완성해주세요.

제한 사항

  • 0 < num1 $\leq$ 100
  • 0 < num2 $\leq$ 100

def solution(num1, num2):

문제7. 두 수의 나눗셈

url: https://school.programmers.co.kr/learn/courses/30/lessons/120810

문제 설명

정수 num1, num2가 매개변수 주어질 때, num1를 num2로 나눈 값에 1,000을 곱한 후 정수 부분을 return 하도록 soltuion 함수를 완성해주세요.

제한 사항

  • 0 < num1 $\leq$ 100
  • 0 < num2 $\leq$ 100

코드 및 설명

def solution(num1, num2):
    return int(num1/num2 * 1000)
