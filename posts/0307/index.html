문제1. 아이스 아메리카노

url: https://school.programmers.co.kr/learn/courses/30/lessons/120819

문제 설명

머쓱이는 추운 날에도 아이스 아메리카노만 마십니다. 아이스 아메리카노는 한잔에 5,500원입니다. 머쓱이가 가지고 있는 돈 money가 매개변수로 주어질 때, 머쓱이가 최대로 마실 수 있는 아메리카노의 잔 수와 남는 돈을 순서대로 담은 배열을 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • 0 < money $\leq$ 1,000,000

입출력 예

money result
5,500 [1,0]
15,000 [2,4000]

코드 및 설명

def solution(money):
    return [money//5500, money%5500]

몫과 나머지 구하는 문제.

문제2. 짝수 홀수 개수

url: https://school.programmers.co.kr/learn/courses/30/lessons/120824

문제 설명

정수가 담긴 리스트 num_list가 주어질 때, num_list의 원소 중 짝수와 홀수의 개수를 담은 배열을 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • 1 $\leq$ num_list의 길이 $\leq$ 100
  • 0 $\leq$ num_list의 원소 $\leq$ 1,000

입출력 예

numbers result
[1,2,3,4,5] [2,3]
[1,3,5,7] [0,4]

코드 및 설명

def solution(num_list):
    return [len([i for i in num_list if i % 2 == 0]), len([i for i in num_list if i % 2 == 1])]

이렇게 풀었는데 다른 사람 풀이에서 인덱스를 이용해서 한 방법이 있었다.

answer[0,0]
for n in num_list:
    answer[n % 2] += 1
return answer

이 방법도 신기했다.

문제3. 특정 문자 제거하기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120826

문제 설명

문자열 my_string과 문자 letter이 매개변수로 주어집니다. my_string에서 letter를 제거한 문자열을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ my_string의 길이 $\leq$ 100
  • letter은 길이가 1인 영문자입니다.
  • my_string과 letter은 알파벳 대소문자로 이루어져 있습니다.
  • 대문자와 소문자를 구분합니다.

입출력 예

my_string letter result
“abcdef” “f” “abcde”
“BCBdbe” “B” “Cdbe”

코드 및 설명

def solution(my_string, letter):
    return my_string.replace(letter,'')

replace를 사용하여 해당 문자를 없앴다.

문제4. 배열 두배 만들기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120809

문제 설명

정수 배열 numbers가 매개변수로 주어집니다. numbers의 각 원소에 두배한 원소를 가진 배열을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • -10,000 $\leq$ numbers의 원소 $\leq$ 10,000
  • 1 $\leq$ numbers의 길이 $\leq$ 1,000

입출력 예

numbers result
[1,2,3,4,5] [2,4,6,8,10]
[1,2,100,-99,1,2,3] [2,4,200,-198,2,4,6]

코드 및 설명

def solution(numbers):
    return [i*2 for i in numbers]

python으로 하니까 코드가 엄청 짧아졌다.

문제5. 점의 위치 구하기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120841

문제 설명

사분면은 한 평면을 x축과 y축을 기준으로 나눈 네 부분입니다. 사분면은 아래와 같이 1부터 4까지 번호를매깁니다.
x 좌표 (x, y)를 차례대로 담은 정수 배열 dot이 매개변수로 주어집니다. 좌표 dot이 사분면 중 어디에 속하는지 1, 2, 3, 4 중 하나를 return 하도록 solution 함수를 완성해주세요.

제한 사항

  • dot의 길이 = 2
  • dot[0]은 x좌표를, dot[1]은 y좌표를 나타냅니다
  • -500 $\leq$ dot의 원소 $\leq$ 500
  • dot의 원소는 0이 아닙니다.

입출력 예

dot result
[2,4] 1
[-7,9] 2

코드 및 설명

def solution(dot):
    if dot[0] > 0:
        if dot[1] > 0:
            answer = 1
        else:
            answer = 4
    else:
        if dot [1] > 0:
            answer = 2
        else:
            answer = 3
    return answer

더 간단한 방법이 있을 것 같은데 잘 모르겠어서 일단 이렇게 코드를 짜 봤다.
제일 인상적이었던 다른 사람 코드

def solution(dot):
    quad = [(3,2),(4,1)]
    return quad[dot[0] > 0][dot[1] > 0]

dot[0] > 0, 즉 x좌표가 0 이상일 경우 true로 quad[1] = (4,1)로 들어간다. 그리고 dot[1] > 0인 경우 quad[][1]로 들어간다. 이렇게 해서 각 4분면을 적절히 찾을 수 있도록 했다.
