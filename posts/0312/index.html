문제1. 가까운 수

url: https://school.programmers.co.kr/learn/courses/30/lessons/120890

문제 설명

정수 배열 array와 정수 n이 매개변수로 주어질 때, array에 들어있는 정수 중 n과 가장 가까운 수를 return 하도록 solution 함수를 완성해주세요.

코드 및 설명

def solution(array, n):
    array.sort()
    abs_array = [abs(i - n) for i in array]
    return array[abs_array.index(min(abs_array))]

우선 array를 정렬했다. 예제에서는 다 정렬되어 있었는데, 혹시나 테스트 케이스에서는 정렬이 안 됐을 수도 있으니..
그리고 각 원소에서 n을 뺀 값의 절댓값을 넣은 리스트를 새로 만들었다.
그리고 거기서 최솟값이 있는 index가 곧 array에서 n과 가장 가까운 값이 있는 index이다.
array를 오름차순으로 정렬 했기 때문에 가장 가까운 수가 여러 개일 경우 더 작은 수를 return할 수 있다.

문제2. k의 개수

url: https://school.programmers.co.kr/learn/courses/30/lessons/120887

문제 설명

1부터 13까지의 수에서, 1은 1, 10, 11, 12, 13 이렇게 총 6번 등장합니다. 정수 i, j, k가 매개변수로 주어질 때, i부터 j까지 k가 몇 번 등장하는지 return 하도록 solution 함수를 완성해주세요.

코드 및 설명

def solution(i, j, k):
    answer = 0
    for a in range(i, j + 1):
        if str(k) in str(a):
            answer += str(a).count(str(k))
    return answer

str(k)가 str(a) 안에 있을 경우, 안에 있는 횟수를 answer에 더하는 방식으로 했다.
이를 한 줄로 줄이면

def solution(i, j, k):
    return sum([str(a).count(str(k)) for a in range(i, j + 1)])

이렇게 된다. 그런데 실행 시간은 1이 더 빠르다.

문제3. 진료 순서 정하기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120835

문제 설명

외과의사 머쓱이는 응급실에 온 환자의 응급도를 기준으로 진료 순서를 정하려고 합니다. 정수 배열 emergency가 매개변수로 주어질 때 응급도가 높은 순서대로 진료 순서를 정한 배열을 return하도록 solution 함수를 완성해주세요.

코드 및 설명

def solution(emergency):
    return [sorted(emergency, reverse = True).index(i) + 1 for i in emergency]

기존 list에 있는 원소들이, 내림차순으로 정렬한 list에서 몇 번째 index인지 찾는다.

문제4. 이진수 더하기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120885

문제 설명

이진수를 의미하는 두 개의 문자열 bin1과 bin2가 매개변수로 주어질 때, 두 이진수의 합을 return하도록 solution 함수를 완성해주세요.

코드 및 설명

def solution(bin1, bin2):
    return bin(int(bin1, 2) + int(bin2, 2))[2:]

이건 풀이를 잘 모르겠어서 구글에서 다른 사람 풀이를 먼저 봤다ㅠㅠ
python의 int()에서 base를 설정할 수 있음을 알게 됐다.
그리고 bin 함수는 integer 자료형을 이진수 문자열로 바꿔서 반환함을 알게 됐다.
반환되는 문자열이 앞에 ‘0b’가 붙으므로 이를 뺀 2번째 index부터를 반환하도록 하였다.

문제5. 소인수분해

url: https://school.programmers.co.kr/learn/courses/30/lessons/120852

문제 설명

소인수분해란 어떤 수를 소수들의 곱으로 표현하는 것입니다. 예를 들어 12를 소인수 분해하면 2 * 2 * 3 으로 나타낼 수 있습니다. 따라서 12의 소인수는 2와 3입니다. 자연수 n이 매개변수로 주어질 때 n의 소인수를 오름차순으로 담은 배열을 return하도록 solution 함수를 완성해주세요.

코드 및 설명

def solution(n):
    answer = []
    for i in range(2, n + 1):
        if n % i == 0:
            answer.append(i)
            while (n % i == 0):
                n //= i
    return answer

이렇게 풀었는데 for문 안에 while문이 있어서 시간 복잡도가 높다.
다른 사람 풀이를 보니까 처음부터 for문이 아닌 while문을 사용해서 해당 소수로 나누어질 때까지 나눈 후 i에 1을 더해서 다음 숫자에 대해서 나누어지는지 확인했다.

def solution(n):
    answer = []
    i = 2
    while i <= n:
        if n % i == 0:
            n //= i
            if i not in answer:
                answer.append(i)
        else:
            i += 1
    return answer

이게 시간 복잡도가 더 낮다.

문제6. 공 던지기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120843

문제 설명

머쓱이는 친구들과 동그랗게 서서 공 던지기 게임을 하고 있습니다. 공은 1번부터 던지며 오른쪽으로 한 명을 건너뛰고 그다음 사람에게만 던질 수 있습니다. 친구들의 번호가 들어있는 정수 배열 numbers와 정수 K가 주어질 때, k번째로 공을 던지는 사람의 번호는 무엇인지 return 하도록 solution 함수를 완성해보세요.

코드 및 설명

def solution(numbers, k):
    return numbers[(1 + 2 * (k - 1)) - ((1 + 2 * (k - 1)) // len(numbers) * len(numbers)) - 1]

처음에는 numbers[] 이렇게 하지 않고 그냥 안의 값을 반환했는데, 생각해보니까 numbers 안에 있는 번호가 꼭 1,2,3,4 이렇게 오름차순으로 나열돼 있다는 보장이 없다는 걸 깨달았다.
numbers[]를 추가하니까 모든 테스트 케이스를 통과했다.
다른 사람 풀이 보니까 numbers[2 * (k - 1) % len(numbers)]를 해도 된다.

문제7. 7의 개수

url: https://school.programmers.co.kr/learn/courses/30/lessons/120912

문제 설명

머쓱이는 행운의 숫자 7을 가장 좋아합니다. 정수 배열 array가 매개변수로 주어질 때, 7이 총 몇 개 있는지 return 하도록 solution 함수를 완성해보세요.

코드 및 설명

def solution(array):
    return sum([str(i).count('7') for i in array])

이건 오늘 푼 문제2와 비슷했다.
다른 사람 풀이 보니까 그냥

def solution(array):
    return str(array).count('7')

이렇게 array를 string 형으로 바꿔서 거기에 직접 count를 적용해도 된다.

문제8. 컨트롤 제트

url: https://school.programmers.co.kr/learn/courses/30/lessons/120853

문제 설명

숫자와 “Z"가 공백으로 구분되어 담긴 문자열이 주어집니다. 문자열에 있는 숫자를 차례대로 더하려고 합니다. 이 때 “Z"가 나오면 바로 전에 더했던 숫자를 뺀다는 뜻입니다. 숫자와 “Z"로 이루어진 문자열 s가 주어질 때, 머쓱이가 구한 값을 return 하도록 solution 함수를 완성해보세요.

코드 및 설명

def solution(s):
    answer = 0
    list_s = list(s.split(' '))
    for i in s:
        if i == 'Z':
            list_s.remove(list_s[list_s.index('Z') - 1])
            list_s.remove('Z')
    return sum(int(i) for i in list_s)

우선 문자열을 공백 기준으로 잘라서 list로 만든 후에, list에서 ‘Z’를 만나면 그 전 원소와 같이 삭제하는 걸로 했다.

문제9. 잘라서 배열로 저장하기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120913

문제 설명

문자열 my_str과 n이 매개변수로 주어질 때, my_str을 길이 n씩 잘라서 저장한 배열을 return하도록 solution 함수를 완성해주세요.

코드 및 설명

def solution(my_str, n):
    answer = []
    for i in range (0, len(my_str) // n * n, n):
        answer.append(my_str[i:i + n])
    if len(my_str) % n != 0:
        answer.append(my_str[i+n:])
    return answer

문자열을 자를 때 end에 index를 넘은 값이 나오면 안 될 줄 알아서 이렇게 나눠서 구했는데, 다른 사람 풀이 보니까 나눌 필요 없이 그냥

def solution(my_str, n):
    return [my_str[i:i+n] for i in range(0, len(my_str), n)]

이렇게 해도 괜찮았다..

문제10. 문자열 계산하기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120902

문제 설명

my_string은 “3 + 5"처럼 문자열로 된 수식입니다. 문자열 my_string이 매개변수로 주어질 때, 수식을 계산한 값을 return 하는 solution 함수를 완성해주세요.

코드 및 설명

def solution(my_string):
    answer = 0
    list_my_string = list(my_string.split(' '))
    answer = int(list_my_string[0])
    for i in range(0, len(list_my_string)):
        if list_my_string[i] == '+':
            answer += int(list_my_string[i + 1])
        elif list_my_string[i] == '-':
            answer -= int(list_my_string[i + 1])
    return answer

처음에는 i를 range()로 하지 않고 i in list_my_string으로 해서 list_my_string[list_my_string.index(i) + 1] 이렇게 index를 구하도록 했는데, 이랬더니 테스트 케이스 반이 실패했다.
테스트 케이스를 추가해서 확인하니까 index를 찾을 때, 해당 문자가 여러 개 있을 경우 첫 번째 index를 돌려주므로 그 다음 연산이 안 됐다.
그래서 index를 직접 도는 방법으로 했다.
다른 사람 풀이가 신기한 게 많다..
특히 eval? 이게 뭔지 찾아봐야겠다.

문제11. 구슬을 나누는 경우의 수

url: https://school.programmers.co.kr/learn/courses/30/lessons/120840

문제 설명

머쓱이는 구슬을 친구들에게 나누어주려고 합니다. 구슬은 모두 다르게 생겼습니다. 머쓱이가 갖고 있는 구슬의 개수 balls와 친구들에게 나누어 줄 구슬 개수 share이 매개변수로 주어질 때, balls개의 구슬 중 share개의 구슬을 고르는 가능한 모든 경우의 수를 return 하는 solution 함수를 완성해주세요.

코드 및 설명

def factorial(n):
    mult = 1
    for i in range(1, n + 1):
        mult *= i
    return mult

def solution(balls, share):
    return factorial(balls) // factorial(share) // factorial(balls - share)

combination 구하는 문제니까 factorial을 별도의 함수로 구현 후 그걸 적용했다.
알고보니 math 라이브러리 안에 comb라는 combination 함수가 있다..

문제12. 삼각형의 완성조건 (2)

url: https://school.programmers.co.kr/learn/courses/30/lessons/120868

문제 설명

선분 세 개로 삼각형을 만들기 위해서는 다음과 같은 조건을 만족해야 합니다.

  • 가장 긴 변의 길이는 다른 두 변의 길이의 합보다 작아야 합니다.

삼각형의 두 변의 길이가 담긴 배열 sides이 매개변수로 주어집니다. 나머지 한 변이 될 수 있는 정수의 개수를 return하도록 solution 함수를 완성해주세요.

코드 및 설명

def solution(sides):
    return 2 * min(sides) - 1

구해야하는 새로운 변을 x라고 할 때,

  1. x가 가장 긴 변일 경우,
  • max(sides) < x < max(sides) + min(sides)
    개수: min(sides) - 1
  1. x가 가장 긴 변이 아닐 경우,
  • max(sides) < min(sidex) + x
  • x < max(sides)
    개수: min(sides) - 1
  1. x가 가장 긴 변이랑 같을 경우
    개수: 1

그래서 이걸 다 더하면 min(sides) * 2 - 1 이 된다.

문제13. 외계어 사전

url: https://school.programmers.co.kr/learn/courses/30/lessons/120869

문제 설명

PROGRAMMERS-962 행성에 불시착한 우주비행사 머쓱이는 외계행성의 언어를 공부하려고 합니다. 알파벳이 담긴 배열 spell과 외계어 사전 dic이 매개변수로 주어집니다. spell에 담긴 알파벳을 한번씩만 모두 사용한 단어가 dic에 존재한다면 1, 존재하지 않는다면 2를 return하도록 solution 함수를 완성해주세요.

코드 및 설명

def solution(spell, dic):
    dic_new = [i for i in dic if len(i) == len(spell)]
    for j in dic_new:
        for a in spell:
            j = j.replace(a, "", 1)
        if j == '':
            return 1
    return 2

이중 for문이라 시간 복잡도가 안 좋다.
다른 사람 풀이 보니까 집합 자료형을 사용한 풀이가 있었다.

문제14. 모스부호 (1)

url: https://school.programmers.co.kr/learn/courses/30/lessons/120838

문제 설명

머쓱이는 친구에게 모스부호를 이용한 편지를 받았습니다. 그냥은 읽을 수 없어 이를 해독하는 프로그램을 만들려고 합니다. 문자열 letter가 매개변수로 주어질 때, letter를 영어 소문자로 바꾼 문자열을 return 하도록 solution 함수를 완성해보세요. 모스부호는 다음과 같습니다.

코드 및 설명

def solution(letter):
    morse = { 
        '.-':'a','-...':'b','-.-.':'c','-..':'d','.':'e','..-.':'f',
        '--.':'g','....':'h','..':'i','.---':'j','-.-':'k','.-..':'l',
        '--':'m','-.':'n','---':'o','.--.':'p','--.-':'q','.-.':'r',
        '...':'s','-':'t','..-':'u','...-':'v','.--':'w','-..-':'x',
        '-.--':'y','--..':'z'
    }
    new_letter = letter.split(' ')
    return ''.join([morse[i] for i in new_letter])

먼저 공백 기준으로 문자열을 분리한 후, 각 모스 부호에 해당하는 문자를 list에 넣어서 join으로 문자열을 만들었다.
