문제1. 배열의 평균값

url: https://school.programmers.co.kr/learn/courses/30/lessons/120817

문제 설명

정수 배열 numbers가 매개변수로 주어집니다. numbers의 원소의 평균값을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 0 $\leq$ numbers의 원소 $\leq 1,000
  • 1 $\leq$ numbers의 길이 $\leq$ 100
  • 정답의 소수 부분이 .0 또는 .5인 경우만 입력으로 주어집니다.

코드 및 설명

def solution(numbers):
    return sum(numbers) / len(numbers)

sum을 구한 후, 해당 배열의 길이로 나눴다.

문제2. 양꼬치

url: https://school.programmers.co.kr/learn/courses/30/lessons/120830

문제 설명

머쓱이네 양꼬치 가게는 10인분을 먹으면 음료수 하나를 서비스로 줍니다. 양꼬치는 1인분에 12,000원, 음료수는 2,000원입니다. 정수 n과 k가 매개변수로 주어졌을 때, 양꼬치 n인분과 음료수 k개를 먹었다면 총얼마를 지불해야 하는지 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • 0 < n < 1,000
  • n / 10 ≤ k < 1,000
  • 서비스로 받은 음료수는 모두 마십니다.

코드 및 설명

def solution(n, k):
    return 12000 * n + (k - n//10) * 2000

10인분마다 음료수 1개 무료이므로 k에서 n//10(n을 10으로 나눈 몫)을 뺀 값에 2000을 곱한다.

문제4. 중앙값 구하기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120811

문제 설명

중앙값은 어떤 주어진 값들을 크기의 순서대로 정렬했을 때 가장 중앙에 위치하는 값을 의미합니다. 예를 들어 1, 2, 7, 10, 11의 중앙값은 7입니다. 정수 배열 array가 매개변수로 주어질 때, 중앙값을 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • array의 길이는 홀수입니다.
  • 0 < array의 길이 < 100
  • -1,000 < array의 원소 < 1,000

코드 및 설명

def solution(array):
    return sorted(array)[(len(array) - 1) // 2]

array.sort()를 하면 array를 정렬하고, list를 따로 return하지 않는다.
따라서, list를 return하는 sorted()를 사용하여 직접 가운데 index를 반환했다.
다른 사람 풀이를 보니까 // 연산이 몫을 구하는 연산이므로 굳이 len(array) - 1을 할 필요 없이 그냥 len(array)//2를 해도 된다.

문제5. 피자 나눠 먹기 (1)

url: https://school.programmers.co.kr/learn/courses/30/lessons/120814

문제 설명

머쓱이네 피자가게는 피자를 일곱 조각으로 잘라 줍니다. 피자를 나눠먹을 사람의 수 n이 주어질 때, 모든 사람이 피자를 한 조각 이상 먹기 위해 필요한 피자의 수를 return 하는 solution 함수를 완성해보세요.

제한 사항

  • 1 $\leq$ n $\leq$ 100

코드 및 설명

def solution(n):
    return n // 7 if n % 7 == 0 else n // 7 + 1

def solution(n):
    return (n + 6) // 7

코드가 좀 더 직관적인 건 1.
2는 더 간단하게 줄이는 방법.

문제6. 피자 나눠 먹기 (2)

url: https://school.programmers.co.kr/learn/courses/30/lessons/120815

문제 설명

머쓱이네 피자가게는 피자를 여섯 조각으로 잘라 줍니다. 피자를 나눠먹을 사람의 수 n이 매개변수로 주어질 때, n명이 주문한 피자를 남기지 않고 모두 같은 수의 피자 조각을 먹어야 한다면 최소 몇 판을 시켜야 하는지를 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • 1 $\leq$ n $\leq$ 100

코드 및 설명

def solution(n):
    for i in range (1, n + 1):
        if i * 6 % n == 0:
            return i

이것도 다른 사람 풀이를 보니까 최대공약수를 사용하는 방법이 있다.
결국 n과 6의 최소공배수를 구하는 문제인데, 최소공배수를 구하는 함수인 lcm는 현재 프로그래머스의 python 버전에서는 제공하지 않나보다.
그래서 최대공약수를 구해서 (n * 6) // math.gcd(n, 6)(여기까지가 최소 공배수) // 6을 하면 피자 개수를 구할 수 있다.

문제7. 피자 나눠 먹기 (3)

url: https://school.programmers.co.kr/learn/courses/30/lessons/120816

문제 설명

머쓱이네 피자가게는 피자를 두 조각에서 열 조각까지 원하는 조각 수로 잘라줍니다. 피자 조각 수 slice와 피자를 먹는 사람의 수 n이 매개변수로 주어질 때, n명의 사람이 최소 한 조각 이상 피자를 먹으려면 최소 몇 판의 피자를 시켜야 하는지를 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • 2 $\leq$ slice $\leq$ 10
  • 1 $\leq$ n $\leq$ 100

코드 및 설명

def solution(slice, n):
    return (n + slice - 1) // slice

피자 나눠 먹기 (1)에서 풀었던 두 번째 방법과 비슷하게 풀었다.

문제9. 배열 자르기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120833

문제 설명

정수 배열 numbers와 정수 num1, num2가 매개변수로 주어질 때, numbers의 num1번 째 인덱스부터 num2번째 인덱스까지 자른 정수 배열을 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • 2 $\leq$ numbers의 길이 $\leq$ 30
  • 0 $\leq$ numbers의 원소 $\leq$ 1,000
  • 0 $\leq$ num1 < num2 < numbers의 길이

코드 및 설명

def solution(numbers, num1, num2):
    return numbers[num1:num2 + 1]

python list의 slicing을 아는지 묻는 문제.
slicing할 때, [start : end : step] 이렇게 지정할 수 있는데, end가 end번째 index까지가 아닌, end번째 index 전까지를 의미한다.

문제10. 배열 뒤집기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120821

문제 설명

정수가 들어 있는 배열 num_list가 매개변수로 주어집니다. num_list의 원소의 순서를 거꾸로 뒤집은 배열을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ num_list의 길이 $\leq$ 1,000
  • 0 $\leq$ num_list의 원소 $\leq$ 1,000

코드 및 설명

def solution(num_list):
    return list(reversed(num_list))

sort와 마찬가지로 list.reverse()메소드를 사용하면 list 자체를 reverse order로 바꾸는 것이므로 return 값이 없다.
reversed()를 사용하면 return 값이 있는데, 이를 list()로 list 형으로 바꿔야된다.

다른 사람 풀이 보니까

def solution(num_list):
    return num_list[::-1]

이런 방법도 있다. start, end를 지정하지 않았으니까 list의 처음부터 끝까지 -1 step씩 새로 list를 만드므로, reverse order가 된다.

def solution(my_string):

문제12. 배열의 유사도

url: https://school.programmers.co.kr/learn/courses/30/lessons/120903

문제 설명

두 배열이 얼마나 유사한지 확인해보려고 합니다. 문자열 배열 s1과 s2가 주어질 때 같은 원소의 개수를 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ s1, s2의 길이 $\leq$ 100
  • 1 $\leq$ s1, s2의 원소의 길이 $\leq$ 10
  • s1과 s2의 원소는 알파벳 소문자로만 이루어져 있습니다
  • s1과 s2는 각각 중복된 원소를 갖지 않습니다.

코드 및 설명

def solution(s1, s2):
    return len([i for i in s1 if i in s2])

find를 쓸까 하다가 그냥 in s2로 해도 된다는 걸 알게 돼서 이렇게 했다.
다른 사람 풀이를 찾아보니까 python에 있는 자료형인 집합을 사용했다.
set()로 해당 list를 집합으로 만든 후, &를 사용하여 두 집합의 교집합을 구한 후 그 교집합의 길이를 구했다.

def solution(s1, s2):
    return len(set(s1) & set(s2))

집합 자료형을 잘 몰라서 이 방법은 생각을 못 했다.

문제14. 모음 제거

문제16. 개미 군단

url: https://school.programmers.co.kr/learn/courses/30/lessons/120837

문제 설명

개미 군단이 사냥을 나가려고 합니다. 개미군단은 사냥감의 체력에 딱 맞는 병력을 데리고 나가려고 합니다. 장군개미는 5의 공격력을, 병정개미는 3의 공격력을 일개미는 1의 공격력을 가지고 있습니다. 예를 들어 체력 23의 여치를 사냥하려고 할 때, 일개미 23마리를 데리고 가도 되지만, 장군개미 네 마리와 병정개미 한 마리를 데리고 간다면 더 적은 병력으로 사냥할 수 있습니다. 사냥감의 체력 hp가 매개변수로 주어질 때, 사냥감의 체력에 딱 맞게 최소한의 병력을 구성하려면 몇 마리의 개미가 필요한지를 return하도록 solution 함수를 완성해주세요.

제한 사항

  • hp는 자연수입니다.
  • 0 $\leq$ hp $\leq$ 1000

코드 및 설명

def solution(hp):
    j = hp // 5
    b = (hp - j * 5) // 3
    c = hp - j * 5 - b * 3
    return j + b + c

이렇게 풀었는데, 다른 사람 풀이 보니까 % 사용해서 더 간단하게 풀 수 있다. 뺄셈을 없애고 hp % 5 // 3 이렇게 바꿀 수 있다ㅠㅠ

문제17. 제곱수 판별하기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120909

문제 설명

어떤 자연수를 제곱했을 때 나오는 정수를 제곱수라고 합니다. 정수 n이 매개변수로 주어질 때, n이 제곱수라면 1을 아니라면 2를 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ n $\leq$ 1,000,000

코드 및 설명

import math
def solution(n):
    return 1 if math.sqrt(n) % 1 == 0 else 2

제곱근을 구한 후 1로 나눴을 때의 나머지가 0인 경우, 즉 정수일 경우에 1을 반환하고, 아닐 경우 2를 반환하도록 하였다.
다른 사람 풀이에서는 math에서의 함수인 sqrt를 사용하지 않고, ** 0.5, 즉 0.5 제곱을 해서 제곱근을 구했다.

문제18. 숨어있는 숫자의 덧셈 (1)

url: https://school.programmers.co.kr/learn/challenges/beginner?order=acceptance_desc&page=2

문제 설명

문자열 my_string이 매개변수로 주어집니다. my_string안의 모든 자연수들의 합을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ my_string의 길이 $\leq$ 1,000
  • my_string은 소문자, 대문자 그리고 한자리 자연수로만 구성되어있습니다.

코드 및 설명

def solution(my_string):
    integer = [1, 2, 3, 4, 5, 6, 7, 8, 9]
    return sum([i * my_string.count(str(i)) for i in integer])

다른 사람 풀이를 보니 string 안에 있는 메소드인 isdigit()를 사용할 수도 있다.
isdigit()은 해당 문자열이 숫자로만 구성되어 있는지를 반환하는 메소드이다.

def solution(my_string):
    return sum(int(i) for i in my_string if i.isdigit())

문제19. 숨어있는 숫자의 덧셈 (2)

url: https://school.programmers.co.kr/learn/courses/30/lessons/120864

문제 설명

문자열 my_string이 매개변수로 주어집니다. my_string은 소문자, 대문자, 자연수로만 구성되어있습니다. my_string안의 자연수들의 합을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ my_string의 길이 $\leq$ 1,000
  • 1 $\leq$ my_string 안의 자연수 $\leq$ 1000
  • 연속된 수는 하나의 숫자로 간주합니다.
  • 000123과 같이 0이 선행하는 경우는 없습니다.
  • 문자열에 자연수가 없는 경우 0을 return 해주세요.

코드 및 설명

def solution(my_string):
    sum_num = 0
    my_string += 'x'
    start = 0
    end = 0
    while(end < len(my_string)):
        if my_string[end].isdigit():
            end += 1
        else:
            if start != end:
                sum_num += int(my_string[start: end])
            start = end + 1
            end = start
    return sum_num

이 문제는 레벨 0에서 제일 어려웠던 것 같다..(무려 8점이다..)
처음에는 어떻게 해야 간단하게 풀지?라고 생각을 했는데, 나중에는 그냥 어떻게 풀지?로 바꼈다.
내가 짠 코드를 해석하자면, 우선 start index와 end index를 start, end 변수로 지정을 해둔다.
그리고 my_string의 처음부터 확인해서 숫자가 아닐 경우, start와 end를 모두 1씩 증가 시킨다.
숫자일 경우에는 다음 문자도 숫자인지 확인해야하므로 start는 그대로 두고 end를 1 증가시킨다.
다음 반복문에서 바로 옆 문자도 숫자인지 확인을 하고 만약 숫자일 경우, start는 그대로 두고 end를 증가시킨다. 만약 옆 문자는 숫자가 아니었을 경우, start index와 end index까지의 substring이 나타내는 숫자를 sum_num에 더한다.
한 가지, 더 추가한 부분은 while문이 my_string의 끝까지 확인하는데, 그 끝이 숫자일 경우에는 해당 수를 sum_num에 더하지 못 하고 끝난다.
따라서, 마지막이 숫자일 경우를 없애기 위해 처음에 my_string에 ‘x’라는 임의의 문자를 뒤에 추가시켰다.

문제20. 암호 해독

url: https://school.programmers.co.kr/learn/courses/30/lessons/120892

문제 설명

군 전략가 머쓱이는 전쟁 중 적군이 다음과 같은 암호 체계를 사용한다는 것을 알아냈습니다.

  • 암호화된 문자열 cipher를 주고받습니다.
  • 그 문자열에서 code의 배수 번째 글자만 진짜 암호입니다.

문자열 cipher와 정수 code가 매개변수로 주어질 때 해독된 암호 문자열을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ cipher의 길이 $\leq$ 1,000
  • 1 $\leq$ code $\leq$ cipher의 길이
  • cipher는 소문자와 공백으로만 구성되어 있습니다.
  • 공백도 하나의 문자로 취급합니다.

코드 및 설명

def solution(cipher, code):
    return ''.join(cipher[i - 1] for i in range(code, len(cipher) + 1, code))

더 간단한 풀이는

def solution(cipher, code):
    return cipher[code - 1::code]

이다.
