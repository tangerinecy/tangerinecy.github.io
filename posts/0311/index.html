문제1. 최빈값 구하기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120812

문제 설명

최빈값은 주어진 값 중에서 가장 자주 나오는 값을 의미합니다. 정수 배열 array가 매개변수로 주어질 때, 최빈값을 return 하도록 solution 함수를 완성해보세요. 최빈값이 여러 개면 -1을 return 합니다.

제한 사항

  • 0 < array의 길이 < 100
  • 0 $\leq$ array의 원소 < 1000

코드 및 설명

def solution(array):
    num = [array.count(i) for i in set(array)]
    if num.count(max(num)) > 1:
        return -1
    else:
        return [i for i in array if array.count(i) == max(num)][0]

우선 중복을 없애기 위해 집합형인 set(array)에 있는 원소들에 대해서 각 원소들의 개수를 num list에 넣었다.
그리고 num안에 있는 최댓값의 개수가 1보다 클 경우, 즉 최빈값이 여러 개일 경우 -1을 반환하고, 최빈값이 1개이면 해당 값을 찾아서 반환하도록 했다.
이 문제도 어려웠다. 배점이 6점이었다.

문제2. 가위 바위 보

url: https://school.programmers.co.kr/learn/courses/30/lessons/120839

문제 설명

가위는 2 바위는 0 보는 5로 표현합니다. 가위 바위 보를 내는 순서대로 나타낸 문자열 rsp가 매개변수로 주어질 때, rsp에 저장된 가위 바위 보를 모두 이기는 경우를 순서대로 나타낸 문자열을 return하도록 solution 함수를 완성해보세요.

제한 사항

  • 0 < rsp의 길이 $\leq$ 100
  • rsp와 길이가 같은 문자열을 return 합니다.
  • rsp는 숫자 0, 2, 5로 이루어져 있습니다.

코드 및 설명

def solution(rsp):
    answer = ''
    for i in rsp:
        if i == '0':
            answer += '5'
        elif i == '2':
            answer += '0'
        else:
            answer += '2'
    return answer

그냥 replace를 사용할 경우에는 2를 0으로 바꾸고 0을 5로 바꿀 때, 이전에 2를 0으로 바꿨기 때문에 해당 0도 바꾸게 될 수가 있다.
따라서, 나는 하나하나 확인하고 answer에 문자를 추가하는 방식으로 했는데, 다른 사람 풀이를 보니까 숫자를 다른 문자로 바꾼 후에 그 문자를 다시 이길 수 있는 숫자로 바꾸는 방법이 있었다.
아니면 dictionary를 사용해서,

def solution(rsp):
    d = {'0':'5','2':'0','5':'2'}
    return ''.join(d[i] for i in rsp)

이렇게 하는 방법이 있다.

문제3. 대문자와 소문자

url: https://school.programmers.co.kr/learn/courses/30/lessons/120893

문제 설명

문자열 my_string이 매개변수로 주어질 때, 대문자는 소문자로 소문자는 대문자로 변환한 문자열을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ my_string의 길이 $\leq$ 1,000
  • my_string은 영어 대문자와 소문자로만 구성되어 있습니다.

코드 및 설명

def solution(my_string):
    answer = ''
    for i in my_string:
        if i >= 'a':
            answer += i.upper()
        else:
            answer += i.lower()
    return answer

이것도 다른 사람 풀이를 보니까 훨씬 더 쉬운 방법이 있었다..

def solution(my_string):
    return my_string.swapcase()

이 메소드 기억해둬야겠다.

문제4. 세균 증식

url: https://school.programmers.co.kr/learn/courses/30/lessons/120910

문제 설명

어떤 세균은 1시간에 두배만큼 증식한다고 합니다. 처음 세균의 마리수 n과 경과한 시간 t가 매개변수로 주어질 때 t시간 후 세균의 수를 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ n $\leq$ 10
  • 1 $\leq$ t $\leq$ 15

코드 및 설명

def solution(n, t):
    return n * 2 ** t

다른 사람 풀이에서는 비트 연산을 써서 n « t로도 풀었다.

문제5. n의 배수 고르기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120905

문제 설명

정수 n과 정수 배열 numlist가 매개변수로 주어질 때, numlist에서 n의 배수가 아닌 수들을 제거한 배열을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ n $\leq$ 10,000
  • 1 $\leq$ numlist의 크기 $\leq$ 100
  • 1 $\leq$ numlist의 원소 $\leq$ 100,000

코드 및 설명

def solution(n, numlist):
    return [i for i in numlist if i % n == 0]

문제6. 직각삼각형 출력하기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120823

문제 설명

““의 높이와 너비를 1이라고 했을 때, ““을 이용해 직각 이등변 삼각형을 그리려고합니다. 정수 n 이 주어지면 높이와 너비가 n 인 직각 이등변 삼각형을 출력하도록 코드를 작성해보세요.

제한 사항

  • 1 $\leq$ n $\leq$ 10

코드 및 설명

n = int(input())
star = '*'
for i in range(1, n + 1):
    print(star * i)

*을 변수로 넣을 필요 없이 그냥

for i in range (1, n + 1):
    print('*' * i)

이렇게 해도 된다.

문제7. 가장 큰 수 찾기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120899

문제 설명

정수 배열 array가 매개변수로 주어질 때, 가장 큰 수와 그 수의 인덱스를 담은 배열을 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • 1 $\leq$ array의 길이 $\leq$ 100
  • 0 $\leq$ array 원소 $\leq$ 1,000
  • array에 중복된 숫자는 없습니다.

코드 및 설명

def solution(array):
    return [max(array), array.index(max(array))]

이건 간단해서 설명 패스.

문제8. 배열 회전시키기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120844

문제 설명

정수가 담긴 배열 numbers와 문자열 direction가 매개변수로 주어집니다. 배열 numbers의 원소를 direction방향으로 한 칸씩 회전시킨 배열을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 3 $\leq$ numbers의 길이 $\leq$ 20
  • direction은 “left” 와 “right” 둘 중 하나입니다.

코드 및 설명

def solution(numbers, direction):
    if direction == "right":
        numbers.insert(0, numbers.pop())
    else:
        numbers.append(numbers.pop(0))
    return numbers

문제9. 숫자 찾기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120904

문제 설명

정수 num과 k가 매개변수로 주어질 때, num을 이루는 숫자 중에 k가 있으면 num의 그 숫자가 있는 자리 수를 return하고 없으면 -1을 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • 0 < num < 1,000,000
  • 0 $\leq$ k < 10
  • num에 k가 여러 개 있으면 가장 처음 나타나는 자리를 return 합니다.

코드 및 설명

def solution(num, k):
    answer = str(num).find(str(k))
    return answer + 1 if answer != -1 else -1

어제 풀었던 문제에도 있었던 것 같은데, string 안에 string 있는 걸 있음, 없음을 확인할 때는 그냥 str1 in str2 이런 식으로 해도 된다.
그래서 다른 사람 풀이 보면

def solution(num, k):
    return -1 if str(k) not in str(num) else str(num).find(str(k)) + 1

이렇게 한 줄로 풀었다.

문제10. 문자열 정렬하기 (1)

url: https://school.programmers.co.kr/learn/courses/30/lessons/120850

문제 설명

문자열 my_string이 매개변수로 주어질 때, my_string 안에 있는 숫자만 골라 오름차순 정렬한 리스트를 return 하도록 solution 함수를 작성해보세요.

제한 사항

  • 1 $\leq$ my_string의 길이 $\leq$ 100
  • my_string에는 숫자가 한 개 이상 포함되어 있습니다.
  • my_string은 영어 소문자 또는 0부터 9까지의 숫자로 이루어져 있습니다

코드 및 설명

def solution(my_string):
    return sorted([int(i) for i in my_string if i.isdigit()])

.sort()는 null을 반복해서 sorted()를 사용했다.

문제11. 문자열 정렬하기 (2)

url: https://school.programmers.co.kr/learn/courses/30/lessons/120911

문제 설명

영어 대소문자로 이루어진 문자열 my_string이 매개변수로 주어질 때, my_string을 모두 소문자로 바꾸고 알파벳 순서대로 정렬한 문자열을 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • 0 < my_string 길이 < 100

코드 및 설명

def solution(my_string):
    return ''.join(sorted(my_string.lower()))

그냥 sorted()를 하면 list가 반환돼서 이를 문자열로 만들어야 된다.

문제12. 합성수 찾기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120846

문제 설명

약수의 개수가 세 개 이상인 수를 합성수라고 합니다. 자연수 n이 매개변수로 주어질 때 n이하의 합성수의 개수를 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ n $\leq$ 100

코드 및 설명

def is_prime(num):
    for i in range (2, num):
        if num % i == 0:
            return False
    return True

def solution(n):
    return len([i for i in range (1, n + 1) if not is_prime(i)])

소수인지 확인하는 함수를 따로 만들어서 소수가 아닌 수들의 list의 길이를 return했다.

문제13. 중복된 문자 제거

url: https://school.programmers.co.kr/learn/courses/30/lessons/120888

문제 설명

문자열 my_string이 매개변수로 주어집니다. my_string에서 중복된 문자를 제거하고 하나의 문자만 남긴 문자열을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ my_string $\leq$ 110
  • my_string은 대문자, 소문자, 공백으로 구성되어 있습니다.
  • 대문자와 소문자를 구분합니다.
  • 공백(” “)도 하나의 문자로 구분합니다.
  • 중복된 문자 중 가장 앞에 있는 문자를 남깁니다.

코드 및 설명

def solution(my_string):
    answer = ''
    for i in my_string:
        if i not in answer:
            answer += i
    return answer

다른 사람 풀이를 보니까 dictionary 자료형에서는 중복된 key가 없으므로 이걸 이용한 풀이가 있었다.

문제14. A로 B 만들기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120886

문제 설명

문자열 before와 after가 매개변수로 주어질 때, before의 순서를 바꾸어 after를 만들 수 있으면 1을, 만들 수 없으면 0을 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • 0 < before의 길이 == after의 길이 < 1,000
  • before와 after는 모두 소문자로 이루어져 있습니다.

코드 및 설명

def solution(before, after):
    return 1 if sorted(before) == sorted(after) else 0

결국 둘이 같은 문자를 갖고 있냐는 문제여서 정렬한 후 같은지 비교했다.

문제15. 2차원으로 만들기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120842

문제 설명

정수 배열 num_list와 정수 n이 매개변수로 주어집니다. num_list를 다음 설명과 같이 2차원 배열로 바꿔 return하도록 solution 함수를 완성해주세요.

num_list가 [1, 2, 3, 4, 5, 6, 7, 8] 로 길이가 8이고 n이 2이므로 num_list를 2 * 4 배열로 다음과 같이 변경합니다. 2차원으로 바꿀 때에는 num_list의 원소들을 앞에서부터 n개씩 나눠 2차원 배열로 변경합니다.

제한 사항

  • num_list의 길이는 n의 배 수개입니다.
  • 0 $\leq$ num_list의 길이 $\leq$ 150
  • 2 $\leq$ n < num_list의 길이

코드 및 설명

def solution(num_list, n):
    answer = []
    for i in range(0, len(num_list), n):
        answer.append(num_list[i:i+n])
    return answer

이걸 한 줄로 줄이면

def solution(num_list, n):
    return [num_list[i:i+n] for i in range(0, len(num_list), n)]

이렇게 할 수 있다. 근데 테스트 케이스 돌렸을 때 시간은 1이 더 빠르다. 왜지?

문제16. 팩토리얼

url: https://school.programmers.co.kr/learn/courses/30/lessons/120848

문제 설명

i팩토리얼 (i!)은 1부터 i까지 정수의 곱을 의미합니다. 예를들어 5! = 5 * 4 * 3 * 2 * 1 = 120 입니다. 정수 n이 주어질 때 다음 조건을 만족하는 가장 큰 정수 i를 return 하도록 solution 함수를 완성해주세요.

i! $\leq$ n

제한 사항

  • 0 < n $\leq$ 3,628,800

코드 및 설명

def solution(n):
    mult = 1
    for i in range(1, n + 1):
        mult *= i
        if mult > n:
            return i - 1
    return i

다른 사람 풀이에서 보니까 math 라이브러리에 factorial 함수가 있다.
마지막에 return i를 안 넣었더니 테스트 케이스 2개는 통과 못 했다.
