My first post

March 4, 2023

This is my first post after making the blog. Here is how I built this blog. I used the Gokarna theme from Hugo themes.

Using hugo

I followed the instructions from here.

hugo new site <project_name>
cd <project_name>

Adding the theme

I used github pages for hosting the Webpage. To separate the whole source code and the built code in different repositories, I first deleted the public directory.

rmdir public
git init
git submodule add https://github.com/526avijitgupta/gokarna.git themes/gokarna
git remote add origin <repository_for_whole_code>
git submodule add -b master <repository_for_public_directory> public

Setting the basic configuration

I changed the config.toml file as the basic configuration shown in the installation wiki.

baseURL = "<github.io url>"
defaultContentLanguage = "en"
languageCode = "en-us"
title = "\U0001F34A cyhwang blog"
theme = "gokarna"

Also, I added some code for the menu and Home page.

Make the Website

Testing the Website

hugo serve

and go to http://localhost:1313 you will see your website.

Building the Website

hugo

public folder will be generated.

Push to git repository

First, push the public repository.

cd public
git add .
git commit -m "<commit message>"
git push origin <repository_for_public_directory> <branch_name>

Then, push the whole code.

cd ..
git add .
git commit -m "<commit message>"
git push origin <repository_for_whole_code> <branch_name>

Note

This was my first time using Hugo and making a Website. I’m still working on using this toml and Markdown.
