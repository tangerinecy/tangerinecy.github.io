문제1. 짝수는 싫어요

url: https://school.programmers.co.kr/learn/courses/30/lessons/120813

문제 설명

정수 n이 매개변수로 주어질 때, n 이하의 홀수가 오름차순으로 담긴 배열을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 1 $\leq$ n $\leq$ 100

입출력 예

n result
10 [1,3,5,7,9]
15 [1,3,5,7,9,11,13,15]

코드 및 설명

c++

#include <string>
#include <vector>

using namespace std;

vector<int> solution(int n) {
    vector<int> answer;
    for (int i = 1; i <= n; i+=2)
        answer.push_back(i);
    
    return answer;
}

python

def solution(n):
    return [i for i in range(1, n + 1, 2)]

for문을 사용하여 1 부터 n 이하까지의 홀수를 answer에 push_back 했다.

문제2. 최댓값 만들기 (1)

url: https://school.programmers.co.kr/learn/courses/30/lessons/120847

문제 설명

정수 배열 numbers가 매개변수로 주어집니다. numbers의 원소 중 두 개를 곱해 만들 수 있는 최댓값을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • 0 $\leq$ numbers의 원소 $\leq$ 10,000
  • 2 $\leq$ numbers의 원소 $\leq$ 100

입출력 예

numbers result
[1,2,3,4,5] 20
[0,31,24,10,1,9] 744

코드 및 설명

c++

#include <string>
#include <vector>
#include <algorithm>

using namespace std;

int solution(vector<int> numbers) {
    int answer = 0;
    sort(numbers.rbegin(), numbers.rend());
    answer = numbers[0] * numbers[1];

    return answer;
}

python

def solution(numbers):
    numbers.sort(reverse=True)
    return numbers[0] * numbers[1]

원소 두 개를 곱해 만들 수 있는 최댓값은 가장 큰 수 2개의 곱이다. algorithm 라이브러리에 있는 sort를 사용하여 vector를 내림차순으로 정렬한다. 그리고 맨 첫 두 원소의 곱을 구하면 된다.

문제3. 최댓값 만들기 (2)

url: https://school.programmers.co.kr/learn/courses/30/lessons/120862

문제 설명

정수 배열 numbers가 매개변수로 주어집니다. numbers의 원소 중 두 개를 곱해 만들 수 있는 최댓값을 return하도록 solution 함수를 완성해주세요.

제한 사항

  • -10,000 $\leq$ numbers의 원소 $\leq$ 10,000
  • 2 $\leq$ numbers의 길이 $\leq$ 100

입출력 예

numbers result
[1,2,-3,4,-5] 15
[0,-31,24,10,1,9] 240
[10,20,30,5,5,20,5] 600

코드 및 설명

c++

#include <string>
#include <vector>
#include <algorithm>

using namespace std;

int solution(vector<int> numbers) {
    int answer = 0;
    sort(numbers.begin(), numbers.end());
    if (numbers[0] * numbers[1] > numbers[numbers.size()-1] * numbers[numbers.size()-2])
        answer = numbers[0] * numbers[1];
    else
        answer = numbers[numbers.size()-1] * numbers[numbers.size()-2];
    return answer;
}

python

def solution(numbers):
    numbers.sort()
    return max(numbers[0]*numbers[1], numbers[-1]*numbers[-2])

(1)과 다르게 (2)에서는 음수도 포함된다. 그리고 입출력 예에서 볼 수 있듯이 음수끼리 곱해서 양수 최댓값이 나올 수도 있다. 그래서 생각한 방법은, 음수끼리 곱했을 때 나올 수 있는 최댓값, 양수끼리 곱해서 나올 수 있는 최댓값을 구하면 둘 중 더 큰 값이 답일 것이다.
그래서 (1)과 마찬가지로 algorithm 라이브러리에 있는 sort를 이용하여 answer를 이번에는 올림차순으로 정렬을 한다. 그러면 첫 2개가 음수일 경우엔, 이 둘의 곱이 음수끼리의 곱 중 최댓값이 되고, 마지막 2개가 양수일 경우엔, 이 둘의 곱이 양수끼리의 곱 중 최>댓값이 된다. 첫 2개가 음수가 아니어도, 마지막 2개가 양수가 아니어도 다른 쪽이 최댓값이 나오므로 괜찮다. 이 둘을 비교하여 더 큰 값을 answer에 넣는다.

다른 사람 풀이를 보니 if문을 사용하지 않고

max(numbers[numbers.size()-1] * numbers[numbers.size()-2], numbers[0], numbers[1])

을 하였다.
algorithm 라이브러리에 있는 max 함수를 사용하였다.

문제4. 문자 반복 출력하기

url: https://school.programmers.co.kr/learn/courses/30/lessons/120825

문제 설명

문자열 my_string과 정수 n이 매개변수로 주어질 때, my_string에 들어있는 각 문자를 n만큼 반복한 문자열을 return 하도록 solution 함수를 완성해보세요.

제한 사항

  • 2 $\leq$ my_string 길이 $\leq$ 5
  • 2 $\leq$ n $\leq$ 10
  • “my_string"은 영어 대소문자로 이루어져 있습니다.

입출력 예

my_string n result
“hello” 3 “hhheeellllllooo”

코드 및 설명

c++

#include <string>
#include <vector>

using namespace std;

string solution(string my_string, int n) {
    string answer = "";
    for (int i = 0; i < my_string.size(); i++) {
        for (int j = 0; j < n; j++)
          answer += my_string[i];
    }
    return answer;
}

이중 for문을 사용하였다. 더 간단한 방법을 찾지 못해서 이렇게 풀고 다른 사람 풀이를 봤는데

for (const auto v : my_string) {
  answer += string(n,v);
}

이런 방법이 있었다. string이 생성자로 사용될 수 있어서 string(size_t count, char ch) 을 사용하여, count 만큼의 문자 ch를 가진 문자열을 생성한다. 참고: 모두의 코드 string 생성자에 대해서 더 공부를 해야겠다.

python 풀이

def solution(my_string, n):
    answer = ''
    for i in my_string:
        answer += i * n
    return answer

python으로는 이렇게 풀었는데 다른 사람 풀이 보니까

return ''.join(i*n for i in my_string)

이 방법이 신기했다. 아직 파이썬이 익숙하지 않아서 함수를 잘 모른다..

문제5. 편지

url: https://school.programmers.co.kr/learn/courses/30/lessons/120898

문제 설명

머쓱이는 할머니께 생신 축하 편지를 쓰려고 합니다. 할머니가 보시기 편하도록 글자 한 자 한 자를 가로 2cm 크기로 적으려고 하며, 편지를 가>로로만 적을 때, 축하 문구 message를 적기 위해 필요한 편지지의 최소 가로길이를 return 하도록 solution 함수를 완성해주세요.

제한 사항

  • 공백도 하나의 문자로 취급합니다.
  • 1 $\leq$ message의 길이 $\leq$ 50
  • 편지지의 여백은 생각하지 않습니다.
  • message는 영문 알파벳 대소문자, ‘!’, ‘~’ 또는 공백으로만 이루어져 있습니다.

입출력 예

message result
“happy birthday!” 30
“I love you~” 22

코드 및 설명

c++

#include <string>
#include <vector>

using namespace std;

int solution(string message) {
    int answer = 0;
    answer = message.length() * 2;
    return answer;
}

python

def solution(message):
    return len(message) * 2

string length 구해서 거기에 2를 곱했다.
